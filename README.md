## 📘 How?

- 매주 두 개의 챕터를 할당합니다.
- 맡은 내용을 학습 후 정리한 내용을 커밋 후 발표합니다.
- 공유된 내용을 확인하고 자신이 이해한 내용과 비교해보면서 간단한 코멘트나 피드백을 진행합니다.
